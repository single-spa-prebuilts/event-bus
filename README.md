# @prebuilts/event-bus

The `@prebuilts/event-bus` module is a [single-spa utility microfrontend](https://single-spa.js.org/docs/module-types#utilities) that allows for inter-app communication. It is free to use in software that uses one of various open source licenses, but must be purchased otherwise. Each major version of event-bus must be purchased separately. The code is available publicly in the npm registry and via Github releases. After purchasing a license, it may be downloaded and used in any way.
